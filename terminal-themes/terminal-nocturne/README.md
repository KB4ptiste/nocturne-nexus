# Nocturne Windows Terminal Scheme v1.0

This repository provides the **Nocturne Nexus** color scheme for :contentReference[oaicite:0]{index=0}, matching your existing Nocturne theme in VS Code.  
It allows you (or other users) to apply the exact same palette in the terminal environment.

---

## Files included

- `Nocturne-WindowsTerminal-scheme-v1.0.json` : The color scheme object to insert into Windows Terminal settings.
- `install-NocturneTerminal.ps1` : PowerShell automation script for installing the scheme (optional).
- `Install.md` : Manual + automated installation instructions.
- `README.md` : (this file) introduction and overview.

---

## Why use this

Using the same palette across your editor (VS Code) and terminal helps achieve visual consistency, reduce cognitive load and maintain brand/identity of your work environment.

---

## How to start

1. Review `Install.md` for installation instructions (manual or automated).
2. Choose your preferred method: manual install or run the script.
3. Make sure to back up existing settings in case you need to revert.

---

## Versioning & Updates

- Version: **v1.0**
- If future palette changes are made in the VS Code theme, the Windows Terminal scheme will also be updated, and you may need to update your settings.
- Use Git tags or release notes to track changes.

## Support & Troubleshooting

If you experience issues (e.g., theme not applied, parse errors), see the troubleshooting section in `Install.md`.
